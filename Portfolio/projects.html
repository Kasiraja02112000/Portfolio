Covid 19 Prediction using Deep learning methods


Deep Learning

Role: Developer

Project Description

A website created to check patients' chest x-rays and CT scans to find those who are impacted by COVID-19 in the population. The front-end development tools for this application are HTML and CSS, while the back-end development tools are Python and Flask. Given that the domain of this application is solely deep learning, a deep learning model like VGG16 is constructed, evaluated, and trained using a sufficient dataset to accurately categorize medical images.

My Role

As a Developer, I developed,tested and trained the Deep learning models and Deployed the trained model by using Python Flask. Even I developed front-end for the Web App by uisng HTML and CSS.
